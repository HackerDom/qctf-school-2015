1. Cоздать пользователя school2015 с шеллом, который выводит ответ:
   gcc -x c - <<< 'int main(){printf("QCTF_NOWYOUKNOWHOWSSHENCRYPTIONWORKS"); return 0;}' -o /usr/bin/qctf
   useradd -s /usr/bin/qctf -m school2015
2. Дать права на шелл, чтобы он был доступен только пользователю school2015: 
   chown school2015:school2015 /usr/bin/qctf
   chmod 500 /usr/bin/qctf
2. Скопировать static/authorized_keys в /home/school2015/.ssh, где /home/school2015 — домашний каталог school2015'а
   mkdir /home/school2015/.ssh
   nano /home/school2015/.ssh/authorized_keys
3. Запустить ssh
4. Проверить, что всё работает: ssh -i solver/id school2015@IP
